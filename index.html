Spatial sensitivity comparative analysis for pedestrian detection

An study of spatial sensitivity and comparative analysis of clasificators used for pedestrian detection

Motivación

El problema de la clasificación de peatones es un problema ampliamente estudiado sin embargo dentro de las posibles soluciones siempre es posible encontrar algunos subproblemas a los cuales es importante realizar ajustes necesarios tanto para algunas aplicaciones en particular como para aumentar el conocimiento que se posee sobre el problema.

Sobre el problema

El subproplema tratado aquí busca analizar el las variaciones en el resultado obtenido de un clasificador en la medida en que la ventana de clasificación de este se desplaza en la vecindad de la detección de un peatón (sensibilidad espacial) lo cual permitirá mejorar o plantear nuevos y mejores métodos de post-procesamiento así como también indicar que elementos son característicos en la respuesta de cada clasificador.

Objetivo

Desarrollar una metodología que permita la comparación y un software de pruebas que permita evaluación sistemática de múltiples clasificadores con el fin de encontrar aquel con mejor sensibilidad espacial para minimizar detecciones múltiples en la vecindad del peatón utilizando el set de datos INRIA (N Dalal, 2005)
